# Hand-Wired 8-Bit Computer

An educational computer built entirely from first principles.

Unlike traditional hobby computers, every logic gate is designed and implemented from individual electronic components before being assembled into larger digital systems.

The goal of this project is to understand exactly how computers work—from transistors to a complete CPU.

---

## Features

- Hand-built logic gates
- Custom ALU
- Custom Control Unit
- Registers
- Program Counter
- Instruction Register
- RAM Interface
- Machine Code Execution
- Expandable Architecture
- Fully documented

## Philosophy

This project intentionally avoids using modern microcontrollers or FPGAs.

Everything is built from the ground up so that every signal, every gate, and every instruction is fully understood.
